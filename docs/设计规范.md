# 设计规范 — 高中生学习助手 (Demo)

## 1. 色彩系统

### 主色调 — 淡蓝色系

| Token | 颜色 | 用途 |
|-------|------|------|
| `--primary` | `#3B82F6` (Blue-500) | 主按钮、链接、强调元素 |
| `--primary-foreground` | `#FFFFFF` | 主色上的文字 |
| `--background` | `#F0F9FF` (Sky-50) | 页面背景 |
| `--card` | `#FFFFFF` | 卡片背景 |
| `--secondary` | `#EFF6FF` (Blue-50) | 次要背景 |
| `--muted` | `#DBEAFE` (Blue-100) | 弱化背景 |
| `--accent` | `#BFDBFE` (Blue-200) | 强调背景 |
| `--border` | `#93C5FD` (Blue-300) | 边框 |
| `--ring` | `#3B82F6` | 聚焦环 |

### 学科色

| 学科 | 颜色 | Tailwind |
|------|------|----------|
| 数学 | 蓝色 | `blue-500` |
| 物理 | 绿色 | `emerald-500` |
| 化学 | 紫色 | `violet-500` |

### 状态色

| 状态 | 颜色 | 用途 |
|------|------|------|
| 成功 | Green-500 | 上传完成、分析完成 |
| 警告 | Amber-500 | 注意事项 |
| 错误 | Red-500 | 失败提示 |
| 信息 | Blue-400 | 一般提示 |

## 2. 字体

- **系统字体栈**：`-apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, "Noto Sans SC", sans-serif`
- **等宽字体**：`"JetBrains Mono", "Fira Code", monospace`（代码/公式用）

## 3. 间距

使用 Tailwind 默认间距系统（基于 4px 网格）：
- 页面内边距：`p-6` (24px)
- 卡片内边距：`p-4` (16px)
- 组件间距：`gap-4` (16px)
- 紧凑间距：`gap-2` (8px)

## 4. 圆角

- 卡片：`rounded-xl` (12px)
- 按钮：`rounded-lg` (8px)
- 输入框：`rounded-md` (6px)
- 标签/徽章：`rounded-full`

## 5. 阴影

- 卡片：`shadow-sm` (轻微阴影)
- 悬浮：`shadow-md` (中等阴影)
- 弹窗：`shadow-lg` (较深阴影)

## 6. 响应式断点

| 断点 | 宽度 | 布局 |
|------|------|------|
| 默认 (mobile) | < 768px | 侧边栏变 Sheet 抽屉，内容纵向堆叠 |
| `md` | ≥ 768px | 侧边栏固定 240px，内容区自适应 |
| `lg` | ≥ 1024px | 两栏布局（如知识树 + 详情） |

## 7. 组件状态

每个数据展示组件需覆盖三种状态：
- **Loading** — 骨架屏 (Skeleton)
- **Empty** — 空状态插画 + 引导文案 + 操作按钮
- **Error** — 错误图标 + 错误信息 + 重试按钮

## 8. 交互规范

- 上传文件：支持点击选择 + 拖拽上传
- 加载中：显示进度条或骨架屏
- 删除操作：弹出确认对话框
- 操作结果：Toast 通知（右上角）
- 流式回复：逐字显示 + 光标闪烁

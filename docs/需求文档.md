# 需求文档 — 高中生学习助手 (Demo)

## 1. 项目背景

为高中生打造一款 AI 驱动的学习辅助工具，帮助学生系统地掌握数学、物理、化学三科知识。Demo 阶段聚焦于**教材知识库构建 + AI 智能问答**两大核心能力。

## 2. 目标用户

- **主要用户**：高中学生（高一至高三）
- **使用场景**：课后复习、预习、备考、答疑

## 3. Demo 功能清单

### 3.1 教材上传与解析
- 支持 PDF、Word (.docx)、图片 (拍照) 三种格式上传
- 自动解析文本内容
- 记录教材元信息：学科（数学/物理/化学）、版本（如人教版）、年级

### 3.2 AI 知识分析
- 自动提取教材章节结构
- 逐章提取知识点（层级树状结构）
- 生成每章的内容总结（2-3 句）
- 识别重点知识列表（3-5 条）
- 识别难点知识列表（含详细解释，2-4 条）

### 3.3 知识可视化
- **知识树**：树状层级展示知识点，支持展开/折叠
- **思维导图**：基于知识点生成交互式思维导图（支持缩放、拖拽）
- **章节总结**：卡片式展示每章总结 + 重点 + 难点

### 3.4 AI 智能问答
- 基于已上传教材内容回答学生问题
- 流式打字效果（实时显示回复）
- 上下文感知：自动匹配问题相关的教材知识点
- 对话历史持久化保存（本地浏览器）

### 3.5 数据管理
- 所有数据存储在浏览器本地（IndexedDB）
- 支持删除教材（级联删除关联数据）
- 支持导出对话记录

## 4. 非功能需求

| 类别 | 要求 |
|------|------|
| 平台 | Web 应用，Windows Chrome/Edge 浏览器 |
| 数据存储 | 纯本地（浏览器 IndexedDB），不上传云端 |
| UI 风格 | 简洁直观，淡蓝色主色调 |
| 响应式 | 适配桌面、平板、手机 |
| 性能 | 页面加载 < 3 秒，聊天首字响应 < 5 秒 |
| 安全 | API Key 仅存服务端，不暴露给前端 |

## 5. 不在 Demo 范围内的功能

以下功能留待正式版：
- 云端数据同步
- 多用户账号系统
- 试卷拍照上传与分析
- 联网搜索公式
- 个性化学习方案生成
- 多设备数据同步
